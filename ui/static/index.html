HBD Thermal Flex 그래픽 테스트 패널

FastAPI 백엔드(`/schemas`, `/simulate`, `/optimize`)와 직접 연동되는 브라우저 기반 실험 도구입니다. 샘플 그래프를 불러와 파라미터를 조정한 뒤 시뮬레이션 또는 최적화를 실행해 보세요. 모든 요청은 순수 JSON API를 사용하므로 동일한 입력을 CLI나 다른 클라이언트에서도 재사용할 수 있습니다.

1. 스키마 탐색

`/schemas` 엔드포인트에서 제공하는 장치 파라미터 스키마 목록입니다. 각 항목을 선택하면 세부 속성을 확인할 수 있습니다. 

스키마를 불러오는 중...

2. PlantGraph JSON

엔진에 제출할 장치-스트림 그래프입니다. 좌측 드롭다운에서 샘플을 선택하거나 직접 JSON을 수정해 보세요.

3. RunCase 정의

운전 모드, 목적 함수, 경계/제약 조건 등을 설정합니다. 기본값은 CCPP 예제를 기준으로 구성되어 있습니다.

mode: simulate
